# 课程数学建模

## 线性规划

线性规划方法用于解决简单优化问题，建立优化问题的数学模型，有三个方面的要素需要确定

* 决策变量
* 目标函数
* 约束条件

如果用数学公式来表示，如下所示
$$
\begin{aligned}
&\min _{x} z=f(x) &(1) \\
&\text { s.t. } \quad g_{i}(x) \le 0 \ (i=1, 2, 3, ..., m) &(2)
\end{aligned}
$$

其中(1)和(2)组成的模型属于约束优化，如果只有(1)就是**无约束优化**，如果目标函数$f(x)$和约束条件中的$g_{i}(x)$都是线性函数，那么这个模型用的方法就是线性规划

建立线性规划模型有三个基本步骤

1. 找到决策变量，用代数符号表示
2. 找出问题中所有约束，写出决策变量的线性方程或线性不等式
3. 找到模型的目标，建立线性目标函数

## 非线性规划

在优化模型中，目标函数或约束条件中至少有一个是非线性函数时的最优化问题叫做非线性规划问题，现实中大部分问题其实是非线性规划问题。

非线性规划的建模思路和线性规划模型类似，但是在非线性函数的处理上需要更加灵活

## 回归模型

**问题**

回归分析是数据分析中最基础也是最重要的分析工具，绝大多数的数据分析问题，都可以使用回归的思想来解决。回归分析的任务就是，通过研究自变量X和因变量Y的相关关系，尝试去解释Y的形成机制，进而达到通过x去预测Y的目的。

常见的回归分析有五类：线性回归、01回归、定序回归、计数回归和生存回归，其划分的依据是因变量Y的类型。本讲我们主要学习线性回归。
