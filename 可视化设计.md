## 代码逻辑

Matplotlib涉及的逻辑基本上跟我们自己绘画的流程是一样的。

- 找一张纸或者一块画布；
- 调色等一系列前戏；
- 作画；

比方说，一般我们画图的时候应该是这么一个流程

第一步，产生一幅图

```python
fig=figure()
```

第二步，建立坐标系

```python
ax=fig.add_axes()
```

第三步，画图

```python
ax.plot()
```

第四步，显示

```python
show()
```

创建一块画布

```python
plt.figure
```

同时创建画布和坐标轴，这也是创建子图合并的API

```python
plt.subplot
```
